Headway Includes:

  • Ace Editor - code editor for Custom Code Block & Live CSS Editor
  • FitVids - Video resizing when Headway responsive grid is turned on
  • FlexSlider - Slider Block
  • HoverIntent - Hover over navigation block parent item for a certain period of time before dropdown triggers
  • Masonry - pinboard masonry layout
  • SelectNav - mobile select nav menu
  • Stellar - parallax feature (v3.8)
  • StickyKit - sticky wrapper feature (v3.8)
  • Superfish - dropdowns in the navigation block

Headway Blocks

Headway specific plugins to use in grid and design mode are referred to as blocks.

Headway Templates

Headway templates contain custom designs, the templates and child themes aren't incompatible.

Template vs Child Theme

Use a child theme if you need to make changes to the theme PHP files. Otherwise, your changes will always be overwritten when you update your theme.
